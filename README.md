**Counter App**

A simple and interactive Counter Application built with React and Vite. This project demonstrates the basics of React state management using the useState hook.

**🚀 Features**
➕ Increase the counter value
➖ Decrease the counter value
🔄 Reset the counter to zero
⚡ Fast development with Vite
🎨 Clean and responsive user interface

**🛠️ Technologies Used**
React
Vite
JavaScript (ES6+)
CSS

**📦 Installation**

# Clone the repository:
git clone https://github.com/samadaamir/counter.git
# Navigate to the project directory:
cd counter-app
# Install dependencies:
npm install
# Start the development server:
npm run dev

The application will be available at:

http://localhost:5173

**🏗️ Build for Production**

To create a production build, run:

npm run build

# To preview the production build:

npm run preview

**📖 How It Works**

The application uses React's useState hook to manage the counter value.

Increase increments the counter by 1.
Decrease decrements the counter by 1.
Reset sets the counter back to 0.
